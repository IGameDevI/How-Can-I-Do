# How Can I Use Hex Color Codes On Xcode?

You can use hex color codes two way:
  
- **On Asset Folder:** <br>
Create new color set and open right menu (⌘ + ⌥ + 0) <br>
- **With Code on .swift file** <br>
You can use with color literal <br>
